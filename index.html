Notes App Project

Git Workflow Assignment – Merge Conflict Example

About This Project

This is a simple Notes App created for demonstrating Git branching, merging, and intentionally creating merge conflicts for educational purposes. The project includes:

  • Initial README setup
  • Multiple Git branches
  • Intentional merge conflict scenario
  • Conflict resolution workflow

Repository Structure

The repository contains the following main files:

  • README.md – Project description and branch details
  • index.html – This basic webpage for better repo presentation

Purpose

The purpose of this project is to practice Git concepts including working with branches, merging, creating conflicts, and resolving them manually.
